My Projects

Colour Sorting System

Designed and built an automated colour sorting system featuring:

  • TCS230 colour sensor with ambient light calibration
  • Arduino-compatible microcontroller
  • OLED display for real-time feedback
  • SG90 servo motor for mechanical sorting
  • Custom PCB shield for component integration
  • 3D-printed enclosure and mechanical parts (CAD designed)

The system detects token colours and sorts them into designated containers based on user's choice, complete with detailed documentation and demonstration video.
