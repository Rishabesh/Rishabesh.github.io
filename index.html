My Projects

Automated Color Sorting System

Designed and built a functional color sorting system with the following components and features:

  • TCS230 color sensor for accurate color detection
  • Arduino-compatible microcontroller for system control
  • OLED display for real-time feedback
  • SG90 servo motor for precise sorting mechanism
  • Automatic calibration to ambient lighting conditions
  • Custom PCB shield designed for optimal component integration
  • CAD-designed auto-feed armature, sensor mount, and token enclosure

The system detects the color of tokens, displays the result on the OLED screen, and sorts each token into designated containers based on user selection.
